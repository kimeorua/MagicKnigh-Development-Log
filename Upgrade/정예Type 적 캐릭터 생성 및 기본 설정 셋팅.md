#### 정예Type 적 캐릭터 생성 및 기본 설정 셋팅
+ #### BaseEnemy 블루프린트를 상속 받아 생성함.
+ #### 스테이터스는 체력 100, 체간 100을 기본으로 함
+ #### 체간 감소 GameplayEffect를 적용 하고, 이동 애니메이션을 제작하여 적용 함.
+ #### AI퍼셉션을 적용하고, 근거리Type과 동일하게 탐지 거리를 설정
